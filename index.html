Two of the greatest things in this world are Zombie Flicks and Science Fiction. I was practically raised on them. Both genres of film have a powerful ability to observe and critique society. What's really special is that they can do everything so subtlely. The films make starting a conversation about social issues easier. Just a few of the incredibly important issues which face society are:

  • Racism
  • classism
  • sexism
  • xenophobia
  • humanism

Comparing my site and another's

The site in question

Visually the site is well done. Everything is sectioned well and easy to read. But, the HTML structure is not as well put together.

One of my biggest cretiuqes for the site below, is that nothing is seperated. None of the films and there relevent information is seperated from all the others. Instead everything is in a paragraph tag all in the same scope. Which is strange because he or she was aware enough to use CSS classes.

One thing I really like is the use of a image tag nested within a reference tag. While, it does simply move to another part of there site, that same system can point to any valid url.

A new tag we have not yet seen is the script tag. The tag will run scripts will add functionality to the site. For example, if a user clicks an element that has a specific class, the developer could set all that classes children to be invisable. The Iframe tag seems to allow the user to scroll through another view on a webpage.
